Differences of Two Proportions

  • Variances of independent variables add

\[SE_{\hat{p}_1 - \hat{p}_2} = \sqrt{SE_{\hat{p}_1}^2 + SE_{\hat{p}_2}^2} = \sqrt{\dfrac{p_1 (1-p_1)}{n_1} + \dfrac{p_2 (1-p_2)}{n_2}}\]

  • Even if you’re subtracting two noisy measurements, the uncertainty from both still adds up
    • Think of it like using two shaky rulers: subtracting doesn’t cancel the shakiness — it just combines it
set.seed(1)       # ensures reproducibility
B <- 1000          # number of simulations
n <- 500           # sample size per group
p1 <- 0.5          # true proportion in group 1
p2 <- 0.4          # true proportion in group 2

# Create empty vectors to store simulated means and SDs
mean_x1 <- mean_x2 <- numeric(B)
sd_x1 <- sd_x2 <- numeric(B)

# Loop to simulate samples for both groups
for (i in 1:B) {
  
  # Generate random binary outcomes for group 1 (success = 1, failure = 0)
  x1 <- rbinom(n, size = 1, prob = p1)
  mean_x1[i] <- mean(x1)  # sample proportion for group 1
  sd_x1[i]   <- sd(x1)    # sample SD for group 1
  
  # Repeat for group 2
  x2 <- rbinom(n, size = 1, prob = p2)
  mean_x2[i] <- mean(x2)
  sd_x2[i]   <- sd(x2)
  
}

# Theoretical standard error for a sample proportion
sqrt(p1*(1-p1)/n)
[1] 0.02236068
# Empirical standard error from simulated data
mean(sd_x1) / sqrt(n)
[1] 0.0223613
# Compute 95% confidence bounds around the sampling mean
ub_x1 <- mean(mean_x1) + 1.96 * sqrt(p1*(1-p1)/n)
lb_x1 <- mean(mean_x1) - 1.96 * sqrt(p1*(1-p1)/n)

# Plot histogram of simulated sample proportions
par(mar = c(4.5, 4.5, 1, 1))
hist(mean_x1, breaks = 12, 
     main = "",
     xlab = "Sample Proportion (Group 1)",
     cex.axis = 1.25, cex.lab = 1.5)
abline(v = c(lb_x1, ub_x1), lty = 2, lwd = 2)  # add CI bounds
# Compute simulated differences
diff <- mean_x1 - mean_x2

# Theoretical SE for the difference in proportions
sqrt(p1*(1-p1)/n + p2*(1-p2)/n)
[1] 0.03130495
# Empirical SE estimate (not exact but illustrative)
mean(sd_x1 + sd_x2) / sqrt(n + n)
[1] 0.03129505
# Compute 95% CIs using correct (additive) and incorrect (subtractive) formulas
## correct
ub_diff_1 <- mean(diff) + 1.96 * sqrt(p1*(1-p1)/n + p2*(1-p2)/n)
lb_diff_1 <- mean(diff) - 1.96 * sqrt(p1*(1-p1)/n + p2*(1-p2)/n)

## incorrect
ub_diff_2 <- mean(diff) + 1.96 * sqrt(p1*(1-p1)/n - p2*(1-p2)/n)
lb_diff_2 <- mean(diff) - 1.96 * sqrt(p1*(1-p1)/n - p2*(1-p2)/n)

# Visualize sampling distribution of the difference
hist(diff, breaks = 12, 
     main = "",
     xlab = "Difference in Sample Proportions (p1 - p2)",
     cex.axis = 1.25, cex.lab = 1.5)

# Add both sets of CI bounds
abline(v = c(lb_diff_1, ub_diff_1), lty = 2, lwd = 2)
abline(v = c(lb_diff_2, ub_diff_2), lty = 3, lwd = 2, col = "red")

# legend
legend("topright",
       legend = c("SE1 + SE2 (correct)", "SE1 - SE2 (incorrect)"),
       lwd = 2, lty = c(2, 3),
       col = c("black", "red"),
       bty = "n")
# Check coverage rates for both formulas
mean(ifelse(diff >= ub_diff_1 | diff <= lb_diff_1, 1, 0))
[1] 0.037
mean(ifelse(diff >= ub_diff_2 | diff <= lb_diff_2, 1, 0))
[1] 0.793

t-distributions

  • When the population standard deviation (\(\sigma\)) is unknown (which is most of the time), we rely on the \(t\)-distribution instead of the normal model.
    • The \(t\)-test helps us quantify uncertainty and make inferences about population parameters using sample data
  • We use a t-distribution with (df = n - 1) to model the sample mean when the sample size is small.
    • As the number of observations increases, the degrees of freedom (df) also increase.
    • With larger df, the t-distribution more closely resembles the standard normal distribution.

t-statistic equation

\[ t = \dfrac{x_i - \bar{x}}{ \sigma_x / \sqrt{n}} \]

# few number of observations; many simulations
n <- 5
Nsim <- 10000

# prep loop
rejections_normal <- 0
rejections_t <- 0
t <- list()

# loop
for(i in 1:Nsim){
  
  # simulate data (n = 2)
  set.seed(i)
  x <- rnorm(n)
  
  # calc. t-stat
  tstat <- mean(x) / (sd(x)/sqrt(n))
  
  # normal critical value
  if(abs(tstat) > 1.96){
    rejections_normal <- rejections_normal + 1
  }
  
  # correct t critical value
  if(abs(tstat) > qt(0.975, df = n-1)){
    rejections_t <- rejections_t + 1
  }
  
  # store t-stat
  t[i] <- tstat
  
}

# gather data from loop; convert to dist.
t <- density(unlist(t))

# generate a normal dist.
set.seed(1)
z <- rnorm(10000)

# plot
par(mar = c(4.5, 4.5, 1, 1))
plot(density(z), 
     xlim = c(-4, 4),
     lwd = 2,
     xlab = "Test Statistics", 
     ylab = "Density",
     main = "",
     cex.lab = 1.5, 
     cex.axis = 1.25)
lines(t, lty = 2, lwd = 2)
legend("topleft", 
       legend = c("normal", "t (df = 4)"),
       lwd = 2,
       lty = c(1, 2),
       bty = "n",
       cex = 1.25)
# p-value if we use cv = 1.96 (wrong)
rejections_normal / Nsim
[1] 0.1245
# p-value if we use cv = t* (correct)
rejections_t / Nsim
[1] 0.0515

Chi-Squared Distributions

  • \(\chi\) = the Greek letter for “chi” (pronounced like “kai”)

  • The \(\chi^2\) distribution is a continuous probability distribution

    • If a variable \(Z\) has the standard normal distribution, then \(Z^2\) has the \(\chi^2\) distribution with one degree of freedom

  • If \(Z_1, Z_2,..., Z_k\) are independent standard normal variables, then…

    • \(Z_1^2 + Z_2^2 +...+ Z_k^2\)
    • …has a \(\chi^2\) distribution with \(k\) degrees of freedom

Degrees of Freedom

  • Degrees of freedom (df): number of independent pieces of information used to estimate a statistic.
    • \(df \neq n\)
    • Imagine tossing a coin 100 times – if you get 55 heads, then you know you got 45 tails (1 df)
    • Imagine a traffic light – if not yellow or red, then green (2 df)
  • Calculate the standard deviation of three exam grades
    • \(s = \sqrt{\frac{ \Sigma (x_i - \bar{x}) }{n - 1} }\)
    • If the mean is already known, then you don’t need all of the other values in the sample
    • If \(\bar{x} = 85\) and \(x_1 = 80\) and \(x_2 = 85\), then \(x_3 = 90\) is no longer independent
  • Why it matters
    • Determines the shape of distributions like \(t\), \(F\), and \(\chi^2\).
    • More df - more information - distributions become narrower and more normal.

Chi-Squared Tests

  • This technique is commonly used in two settings:
    • To determine whether a sample is representative of a larger population across several groups
    • To evaluate whether observed data follow a specified theoretical distribution (e.g., normal, geometric, Poisson)
  • Allows us to examine all bins simultaneously, rather than testing one or two categories in isolation

\[ \chi^2 = \sum_i \frac{(O_i - E_i)^2}{E_i} \]

  • Where:
    • \(O_i\) = observed count in category \(i\)
    • \(E_i\) = expected count under the null hypothesis
  • This statistic follows a chi-squared distribution with \((k - 1)\) degrees of freedom, where \(k\) is the number of categories

Chi-Squared Tests: Example 1

  • Consider data from a random sample of 275 jurors in a small county
    • We would like to test whether this sample is racially representative of the eligible population of jurors (i.e., registered voters)
    • \(H_0\): Jurors are randomly selected; differences reflect natural sampling variation

Chi-Squared Tests: Example 2

  • Suppose we analyze daily stock returns from the S&P 500 over a 10-year period to test whether market movements are independent across days

    • “Does knowing whether the market went up or down yesterday help predict what happens today?”

  • If daily market movements are truly independent, the waiting time between positive days should follow a geometric distribution

  • The geometric distribution gives the probability of waiting \(k\) trials to observe the first success

    • \(p\) = probability that a given day is an Up day
    • \(X\) = number of days until the next Up day

\[P(X = k) = (1 - p)^{k - 1}p\]

ANOVA and F-statistics

  • When comparing more than two groups, we might be tempted to run multiple pairwise tests:
    • Group 1 vs. 2, Group 1 vs. 3, Group 2 vs. 3, etc.
  • Problem: Multiple tests increase the chance of false positives
  • Solution: Use a single, holistic test to compare all group means at once

\[ H_0: \mu_1 = \mu_2 = \cdots = \mu_k \quad\quad H_A: \text{At least one mean differs} \]

  • ANOVA: tests whether there is evidence that at least one mean differs
    • Between-group variation (differences in group means)
    • Within-group variation (random noise)
    • Intuition: If group differences are large relative to within-group variation, we reject the null.
  • This produces the F-statistic

ANOVA: Example 1

  • Suppose a statistics department runs three sections (A, B, and C) of an introductory statistics course
    • We want to know whether there are statistically significant differences in the average final exam scores

ANOVA: SSG

  • The sum of squares for the groups (SSG) measures the differences between the group means \(\bar{x}_i\) and the grand mean \(\bar{x}\) like so:

\[ SSG = \sum_{i=1}^k n_i (\bar{x}_i - \bar{x})^2 \]

# grades in each section
a <- c(90, 87, 88, 78)
b <- c(87, 85, 80)
c <- c(95, 93, 90, 88, 85)

# number of grades in each section
na <- length(a)
nb <- length(b)
nc <- length(c)

# average grades in each section
ua <- mean(a)
ub <- mean(b)
uc <- mean(c)

# grand mean
u <- mean(c(a, b, c))

# SSG
SSG <- na*(ua - u)^2 + nb*(ub - u)^2 + nc*(uc - u)^2
SSG
[1] 84.11667

ANOVA: MSG

  • The mean square between groups (MSG) is calculated using both the groups’ degrees of freedom \(df_G\) and the sum of squares between groups SSG:

\[ MSG = \dfrac{1}{k - 1} SSG \]

k <- 3
df1 <- k - 1

MSG <- SSG / df1
MSG
[1] 42.05833

ANOVA: SSE

  • The sum of squared errors (SSE) is the sum of the squares of the differences between each sample’s observations with each respective sample mean
    • The calculation is simplified by first finding the variance \(s_i^2\) of each sample:

\[ SSE = \sum_{i = 1}^k (n_i - 1) s_i^2 \]

# standard errors for each group
sa <- sd(a)
sb <- sd(b)
sc <- sd(c)
sa; sb; sc
[1] 5.315073
[1] 3.605551
[1] 3.962323
# SSE
SSE <- (na - 1)*sa^2 + (nb - 1)*sb^2 + (nc - 1)*sc^2
SSE
[1] 173.55

ANOVA: MSE

\[ MSE = \dfrac{1}{n - k} SSE \]

# MSE
n <- length(c(a, b, c))
MSE <- SSE / (n - k)
MSE
[1] 19.28333

ANOVA: F-stat

  • We now have comparable measures of variability both between the groups (MSG) and within the groups (MSE).
    • If variation among group means is simply random, F is typically close to 1

\[ F = \dfrac{MSG}{MSE} \]

# f-stat
f_stat <- MSG / MSE
